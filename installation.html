Firestorm – Installation Guide

Please follow these conventions when installing and using Firestorm:

  1. Install the source code in a directory named "fs"
  2. Create a subdirectory of "fs" for each database, with a name that is in lowercase letters (or digits) and no longer than 5 characters
  3. Install the METAD database in a directory named "metad" which is not a subdirectory of "fs"
  4. Create a subdirectory of "metad" for each database, with a name that is in lowercase letters (or digits) and no longer than 5 characters. This name should match exactly the corresponding subdirectory of "fs"

For UNIX-style operating systems,

  1. verify that the file fs001122.tar.gz is 61,488 bytes
  2. uncompress it with the command "gunzip fs001122.tar.gz"
  3. verify that the file fs001122.tar is 399,360 bytes
  4. extract the source files with the command "tar -xvf fs001122.tar"
  5. compile the source files with the command "make"

For DOS and Windows-based operating systems,

  1. verify that the file fs001122.zip is 63,140 bytes
  2. extract the source files using PKZIP or WinZip
  3. compile the source files

To install the METAD database on your DOS or Windows system,

  1. verify that the md001126.zip file is 42,612 bytes
  2. extract the files using PKZIP or WinZip
  3. create the METAD database with the command "DPIMP"
